# REQ-014: Commit Application

## Description
Apply commits from source VCS to Git repository with proper file operations.

## Acceptance Criteria
- [ ] Can apply a commit with author, date, and message
- [ ] Handles file additions correctly
- [ ] Handles file modifications correctly
- [ ] Handles file deletions correctly
- [ ] Commit hash is deterministic (for verification)

## Test Cases
1. Apply commit with single file addition
2. Apply commit with multiple files
3. Apply commit with file modification
4. Apply commit with file deletion
5. Commit metadata is preserved
