# REQ-015: Branch and Tag Creation

## Description
Create branches and tags in Git repository to match source VCS structure.

## Acceptance Criteria
- [ ] Can create a branch at a specific commit
- [ ] Can create a tag at a specific commit
- [ ] Can create annotated tags
- [ ] Branches and tags are properly listed

## Test Cases
1. Create branch from HEAD
2. Create branch from specific revision
3. Create lightweight tag
4. Create annotated tag
5. List branches
6. List tags
